Code Smell Analysis on Blocks Programs (Scratch)

Just like text-based programming langauges, blocks programming suffers design smells. The goal for this work is to find out what types of code smells exists in blocks programs. How are they different from text-based counterpart. This is interesting because the findings will be the empirical studies of the end-user developers as opposed to experienced users
